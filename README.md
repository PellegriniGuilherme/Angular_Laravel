# Angular_Laravel
Projeto indique um amigo, back-end desenvolvido em Laravel e front-end desenvolvido em Angular

#### Instalação
#Configurar arquivo .env Laravel com base de dados disponível
_Por favor rodar comandos na ordem_
```
  cd back_end
  composer install
  php artisan migrate
  php artisan db:seed
  
  cd ..
  
  cd front_end
  npm i
  ng serve
```
